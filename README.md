- En JavaScript, hay varios tipos de datos que se utilizan para representar diferentes tipos de valores. Aquí tienes cinco tipos de datos comunes en JavaScript: 
1. **Números (Numbers):**  Representan valores numéricos, ya sean enteros o números de punto flotante. Por ejemplo, 42, 3.14, -10, etc. 
2. **Cadenas de texto (Strings):**  Representan secuencias de caracteres. Se pueden definir utilizando comillas simples ('') o comillas dobles (""). Por ejemplo, "Hola, mundo!". 
3. **Booleanos (Booleans):**  Representan valores verdaderos o falsos. Solo existen dos valores booleanos: true (verdadero) y false (falso). 
4. **Arreglos (Arrays):**  Son estructuras que almacenan una lista ordenada de valores, que pueden ser de cualquier tipo de dato, incluyendo otros arreglos. Se definen utilizando corchetes []. Por ejemplo, [1, 2, 3]. 
5. **Objetos (Objects):**  Son colecciones de propiedades que asocian nombres (claves) con valores. Cada propiedad tiene una clave y un valor asociado. Se definen utilizando llaves {}. Por ejemplo:

```javascript

const persona = {
  nombre: "Juan",
  edad: 30,
  ciudad: "Madrid"
};
```



Estos son solo algunos de los tipos de datos en JavaScript. También hay otros como `undefined`, `null`, `Symbol`, `Date`, etc. JavaScript es un lenguaje dinámico, lo que significa que las variables pueden cambiar de tipo durante la ejecución del programa.
