# Text Elements
 **Números (Numbers):**  Representan valores numéricos, ya sean enteros o números de punto flotante. Por ejemplo, 42, 3.14, -10, etc.  ^pmQEcfVV

**Cadenas de texto 
(Strings):**  Representan
 secuencias de caracteres. 
Se pueden definir utilizando 
comillas simples ('') o comillas
 dobles (""). 
Por ejemplo, "Hola, mundo!". ^W25wZGwd

**Booleanos (Booleans):**  Representan valores verdaderos o falsos. Solo existen dos valores booleanos: true (verdadero) y false (falso).  ^iyXUBudR

 **Arreglos (Arrays):** 
 Son estructuras que
 almacenan una lista ordenada
 de valores, que pueden ser 
de cualquier tipo de dato,
 incluyendo otros arreglos. 
Se definen utilizando corchetes [].
 Por ejemplo, [1, 2, 3].  ^PNUsoisE

. **Objetos (Objects):**  Son colecciones de propiedades que asocian nombres (claves) con valores. Cada propiedad tiene una clave y un valor asociado. Se definen utilizando llaves {} ^OlaDT2Dn
